# react-native-mtp-camera

 get live image from nikon or canon  camera to mobile through usb

## Installation

```sh
npm install react-native-mtp-camera
```

## Usage


```js
import { multiply } from 'react-native-mtp-camera';

// ...

const result = multiply(3, 7);
```
